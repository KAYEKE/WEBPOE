Website

Green Cape Town is a student-led environmental initiative focused on tree planting, waste 
reduction, and sustainability education in Cape Town and beyond. This website serves as the 
organization's digital presence, providing information about projects, facilitating volunteer 
sign-ups, and enabling community engagement.
PROJECT GOALS
✓ Create a professional, responsive website showcasing Green Cape Town's 
environmental work
✓ Facilitate volunteer recruitment and community engagement
✓ Provide information about projects and initiatives
✓ Enable easy contact and enquiry submission
✓ Optimize for search engines (SEO) and accessibility
✓ Demonstrate web development skills including HTML5, CSS3, and JavaScript

DESIGN DECISIONS
Color Palette
• Primary Green: #2ecc71 (Environmental theme)
• Dark Green: #1b5e20 (Accents and buttons)
• Forest Green: #0f9d58 (Background gradient)
• White: #ffffff (Text on dark backgrounds)
• Error Red: #ff6f61 (Form validation errors)
Typography
• Primary Font: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
• Heading Font: Montserrat (sans-serif)
• Base Size: 16px
• Line Height: 1.6 for optimal readability
Layout
• Max Width: 1200px for content containers
• Grid System: CSS Grid for complex layouts, Flexbox for simpler components
• Spacing: Consistent rem-based spacing system
• Breakpoints: 480px (mobile), 820px (tablet), 1024px+ (desktop)
